Emergency Medical Services Unit Database

Database Project for EMS Operations & Emergency Response

Project Overview

This project focuses on managing Emergency Medical Services (EMS) units, ambulances, staff, and emergency incidents using a structured database system.

Main Features

  • EMS Units Management
  • Ambulances and Staff Records
  • Emergency Call Tracking
  • Response Time Analysis
  • Patient Information (Simulation)

Technologies

  • SQL Database
  • ER Diagram Design
  • HTML (Frontend Demo)
